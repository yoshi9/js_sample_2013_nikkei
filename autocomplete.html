html部分 
 
    $('#txt').autocomplete({
        source : 'list.php', 
        delay : 500, 
        minLength : 1,  
        autoFocus : true 
    });


$(document).ready( function() {
$("#ac2").autocomplete({
	source: function(req, resp){
		$.ajax({
		    url: "autocomplete-datasource.php",
		    type: "POST",
		    cache: false,
		    dataType: "json",
		    data: {
		      param1: req.term
		    },
		    success: function(o){
		    	resp(o);
		    },
		    error: function(xhr, ts, err){
		    	resp(['']);
		    }
		  });

	}
});
});
req.term にはユーザーの入力値が渡されます。resp はファンクションで、配列を渡すことでオートコンプリートリストに表示されます。
